# Telegram Drive Chrome Extension

This Chrome extension is a lightweight launcher for Telegram Drive.

## Load Unpacked

1. Open `chrome://extensions`.
2. Enable **Developer mode**.
3. Choose **Load unpacked**.
4. Select this `extension` folder.

## Release ZIP

GitHub Actions packages this folder as `Telegram_Drive_<version>_chrome_extension.zip` and uploads it to each tagged release.

For manual packaging, zip the contents of this folder so `manifest.json` is at the root of the archive.
